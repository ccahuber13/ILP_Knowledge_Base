This article describes how to create events and the differences between creating each type.

To create a new event:

  1. On the main navigation, click the Events tab.
  2. Click the New Event button.
  3. Choose an Event Type then follow the steps in the corresponding procedures below.

Creating Instructor-Led Training events

Instructor-led Training events are multi-user events where the instructor and students can be in a classroom at the same location or online.

To create an Instructor-Led Training event:

  1. Enter an Event Name and select an Event Owner. The Event Owner is the primary contact for the event.
  2. Select a Timezone and select if the event is Online or in a Classroom location.
    • The Timezone determines the 2AM start and 10PM end time of the event.
    • The Online and Classroom options indicate if users are attending online training from different geographic locations, or attending at a physical classroom facility.
  3. Select a Start Date and End Date.
  4. Optional - Select to Start Instructor code early.
    • Starting the instructor code early can be useful for instructors to help prepare for online training, such as becoming familiar with the remote environment or training materials.
    Starting an instructor code early incurs additional costs for each each the code is started early.
  5. Select a Billing Entity to indicate where to send the event invoice.
  6. Optional - Enter a Purchase Order and or External Reference.
    • Purchase Order is not required by ReadyTech but may be required by your organization for billing purposes.
    • External Reference is for your orgaization. For example, you may want to reference an entity in your organizations Learning Management System with a ReadyTech event.
  7. Select an Instructor for the event.
    Select New user to quickly add a new instructor for the event.
  8. Optional - Select an Assistant Instructor.
    • An assistant instructor has the same capabilities to monitor, screen share, and help students as the event Instructor.
  9. Enter Number of Seats for each instructor, assistant instructor, and student attending the event.
  10. Optional - Select a Template. Lab Type and Deployment are autofilled from the Template.
    Templates organize Lab Type, Deployments, and training materials for a course. This is helpful for instructors to more easily select the correct course for an event. See Templates for more information.
  11. Select a Lab Type.
    • Lab Type determines if the event utilizes deployments and images where students connect to a remote system, or is a browser only SaaS Labs training.
  12. Select a Deployment.
  13. Select a Classroom Tool.By default, Classroom Tool is set to LabManager. However, if you don't need a classroom tool, select none from the Classroom Tool drop-down list.
  14. Optional - Select Send email notifications. You can choose from a list of different notification templates.
  15. Click Save

Creating ILT Flex events

An ILT FLex is a flexible instructor led training event that allows you to stop and start the labs between class days.

ILT Flex only works for lab environments hosted in our cloud and require a Course Tempate. Hardware labs are not supported.

To create an ILT Flex event:

  • Enter Basic Information and Billing Information, Instructor and Labs, and choose to Send Notification, similar to creating an Instructor-Led Training event.

Creating Instructor-Led Workshop events

A workshop event is a 4 hour event at 60% of the price of daily labs.

To create an Instructor-Led Workshop event:

  1. Under Basic Information, select an Access Start Time. Access End Time automatically populates your four hour end time.
  2. Continue entering Basic Information and Billing Information, Instructor and Labs, and choose to Send Notification, similar to creating an Instructor-Led Training event.
You can extend workshop events to a full day, or multiple days if needed.

Creating SaaS Lab events

A SaaS Lab event does not require cloud or hardware labs. Instead, instructors and students utilize the Instructor-Led Portal to train with browser based SaaS software.

To create a SaaS labs event:

  • Follow the steps to create an Instructor-Led Training event. When selecting a Lab Type, select Lab Plan, then choose a Lab Plan.

Creating Self-Paced events

Self-Paced training events are single seat events and require a self-paced course template to be defined ahead of time.

To create a Self-Paced event:

  1. Enter an Event Name and select an Event Owner. The Event Owner is the primary contact for the event.
  2. Select a Timezone for the event.
  3. Select a Start Date.
  4. Select a Billing Entity to indicate where to send the event invoice.
  5. Optional - Enter a Purchase Order and or External Reference.
    • Purchase Order is not required by ReadyTech but may be required by your organization for billing purposes.
    • External Reference is for your orgaization. For example, you may want to reference an entity in your organizations Learning Management System with a ReadyTech event.
  6. Optional - Enter the Company the the student is representing.
  7. Enter the students First Name, Last Name, and Email address.
  8. Select a Course Template.
  9. Optional - Choose to Send email notification. If so, Select notifcation template Access Code - Self Paced.
    • To customize your Self-Paced notification templates, see creating and customizing notification templates.
  10. Click Save.

Creating Sales Demo events

A Sales Demo event is used to showcase and demonstrate your technology to another party.

To create a Sales Demo event:

  1. Enter an Event Name and select an Event Owner. The Event Owner is the primary contact for the event.
  2. Select a Timezone for the event.
  3. Select a Start Date.
  4. Select a Billing Entity to indicate where to send the event invoice.
  5. Optional - Enter a Purchase Order.
    • Purchase Order is not required by ReadyTech but may be required by your organization for billing purposes.
  6. Below Sales Rep and Template Information, choose if the Sales Rep is an existing Admin Portal user.
    • If you choose Yes, select the user from the Name drop-down list that appears.
    • If you choose No, continue entering user information.
  7. Optional - Enter the Company the the Sales rep is representing.
  8. Enter the Sales Rep's First Name, Last Name, and Email address.
  9. Select a Course Template.
  10. Optional - Choose to Send email notification. If so, Select notifcation template Event Creation.
    • The access code for the Sales Demo event is emailed to the owner and sales rep twenty-four (24) hours before the start date.
    • To create Sales Demo notification templates, see creating and customizing notification templates.
  11. Click Save.

Creating Instructor Prep events

Instructor prep events are normally a single seat event to allow an instructor to get prepared for a course delivery.

To create an Instructor Prep event:

  • Follow the same general steps for creating an Instructor-Led Training event.
  • Instructor Prep events do not contain assistant instructors. If you'd like to practice with LabManager or ClassManager, you can add an additional access code to a prep event after it has been created.

Creating Image Update events

An Image Update event is normally a single seat event to make updates to the master image.

To create an Image Update event:

  1. Enter an Event Name and select an Event Owner. The Event Owner is the primary contact for the event.
  2. Select a Timezone for the event.
  3. Select a Start Date. The End Date is automatically set to seven (7) days after the start date.
  4. Optional - You can select Edit End Date if you need more than seven (7) days to update your image. However, regular daily rates apply for additional days.
  5. Select a Billing Entity to indicate where to send the event invoice if additional days are added.
  6. Optional - Enter a Purchase Order and or External Reference.
    • Purchase Order is not required by ReadyTech but may be required by your organization for billing purposes.
    • External Reference is for your orgaization. For example, you may want to reference an entity in your organizations Learning Management System with a ReadyTech event.
  7. Number of Seats is set to one (1). To add additional seats, select Edit #of seats.
    • A daily rate applies for additional access codes.
    • If working with multilple access codes in an update event, verify you are capturing the correct code as the master image.
  8. Select a Lab Type, Deployment or Deployment Plan.
  9. Select a Deployment. This is the configuration of the image you want to update.
  10. Optional - Select Yes to Send an email notification. From the Select notification template drop-down list, select Image Update.
    • The Image Update notification includes the access code and is sent to the event owner by default.
  11. Click Save.

Creating Sandbox events

A Sandbox event is one with a list of duration options which can be one month or one year.

To create a Sandbox event:

  1. Enter an Event Name and select an Event Owner. The Event Owner is the primary contact for the event.
  2. Select a Timezone for the event.
  3. Select a Start Date, then select a Duration. The End Date is automatically set based on the chosen duration.
    • A Sandbox event can be one (1) month to twelve (12) months in length.
  4. Select a Billing Entity to indicate where to send the event invoice.
  5. Optional - Enter a Purchase Order and or External Reference.
    • Purchase Order is not required by ReadyTech but may be required by your organization for billing purposes.
    • External Reference is for your orgaization. For example, you may want to reference an entity in your organizations Learning Management System with a ReadyTech event.
  6. Select the primary user of the Sandbox event from the Assigned To drop-down list.
  7. Select a Lab Type, Deployment or Deployment Plan.
  8. Select a Deployment.
    • You can deploy whatever Deployment you want to the Sandbox event once it's created.
  9. By default, Classroom Tool is set to LabManager. However, if you don't need a classroom tool, select none from the Classroom Tool drop-down list.
  10. Optional - Select Yes to Send an email notification. For Sandbox notifications, you can use the Event Creation notification.
  11. Click Save.
