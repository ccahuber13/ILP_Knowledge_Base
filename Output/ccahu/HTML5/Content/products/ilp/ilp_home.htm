Frequently Asked Questions

What browser can I use with ReadyTech?

Users will need a browser that supports HTML5. See the following list of browsers and the minimum versions that are supported.

  • Internet Explorer 10 or higher
  • Firefox 15 or higher
  • Opera 12 or higher
  • Chrome 16 or higher
  • Safari or higher
  • Edge

How much bandwidth do I need?

The default ReadyTech Viewer requires at least 250KB/s of download speed for a single user. You can check your available bandwidth using the Pre- check feature on our portal pages.

What if the test shows I have insuffecient bandwidth?
  • If you are using a wireless connection, try switching to a hard wired connection if possible
  • Make sure you are not downloading / uploading or streaming video / music.

My latency is above 200ms, what can I do?

Latency is the time it takes for data to transfer from point A to B in this case from your computer to ReadyTech’s data center. The route the data takes is controlled by your ISP and the various back bone internet providers between you and ReadyTech and there is not much you can do to change this. That said here are some things worth trying:

  • If connecting from home, make sure to try a hardwired connection
  • Check if anything or anyone on your network is streaming movies, music, or large files.

Can users connect with a wireless connection?

Hardwired connections are strongly recommended. Wireless signal quality can vary depending on the location, wireless access point strength, and the number of wireless access points available. The required bandwidth still applies to wireless network connection.

A wireless connection is used, use Pre-check to determine network bandwidth. In a single classroom location with multiple students, verify with the local network administrator that there are enough available wireless access points to handle the number of users that will be connecting to ReadyTech.

How much bandwidth is required for multiple students in a single location?

For a class being taught in a single location, each user will need 250KB/s (download) of bandwidth for each user. Use our bandwidth calculator to check your expected network performance with multiple students.

  1. On the Instructor-Led Portal home page, click Pre-check .
  2. Click Start and run the bandwidth check.
  3. Below the download speed check, click Use the bandwidth calculator.
  4. On the bandwidth calculator window, enter the number of users and click Calculate.

What can I do if the bandwidth test fails?
  • If you are using a wireless connecting switch to a wired connect if possible.
  • Make sure you are not downloading or uploading large files in the background, or streaming video and music.
  • If you are at work or in an office location, verify your bandwidth. If is below our requirements, check with your network administrator.
  • Check with your network administrator if there is any heavy network activity that can affect your bandwidth.
  • If you are connected to a VPN, disconnect from the VPN and connect to our service without using a VPN connection.
