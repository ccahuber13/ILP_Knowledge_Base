Frequently asked questions

What browser can I use with ReadyTech web portals?

See the following list of supported HTML5 browsers:

  • Internet Explorer 10 or higher
  • Firefox 15 or higher
  • Opera 12 or higher
  • Chrome 16 or higher
  • Safari or higher
  • Edge

If HTML5 is not supported, try using a different browser or upgrading your current browser.

How much bandwidth is needed to use the ILP?

A single ILP user needs at least 250 KBps of download bandwidth. If multiple users are connecting to the ILP from the same network location, the network needs to support download bandwidth for each active user.

Multiple users on a single network must be taken into consideration as well. To learn how to calculate bandwidth for multiple students in a single location, see calculating bandwidth for multiple students.

What can I do if the pre-check test shows insufficient bandwidth?

If you are connecting from a home network, contact your internet service provider to confirm the download bandwidth for your data plan. If you are connecting from an office or work location, check with the network administrator if an issue with the network is causing bandwidth issues.

The pre-check test shows my latency is above 200ms, what can I do?

Latency is the time it takes for data to move from your local system to a ReadyTech data center. Latency can be highly dependent on multiple factors such as internet service provider, weather, location, and network traffic.:

  • Switch from Wireless connection to a hardwired connection, or vice versa.
  • Verify no large files or videos are being streamed on the network.
  • If you are connected to a VPN, try disconnecting the VPN.
Can I use the Instructor-Led Portal with a wireless connection?

Hardwired connections are strongly recommended. Wireless signal quality can vary depending on the location, wireless access point strength, and the number of wireless access points available. The required download bandwidth (250 KBps) still applies to wireless network connections.

How much bandwidth is required for multiple students in a single location

For a class being taught in a single location, each user will need 250KB/s (download) of bandwidth for each user. Use our bandwidth calculator to check your expected network performance with multiple students.

  1. On the Instructor-Led Portal home page, click Pre-check .
  2. Click Start and run the bandwidth check.
  3. Below the download speed check, click Use the bandwidth calculator.
  4. On the bandwidth calculator window, enter the number of users and click Calculate.
