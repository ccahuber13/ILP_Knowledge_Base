Viewing system logs

Use System log… to access the log for a remote lab. The system log is useful to check for errors or the status of the following:

  • Rebooting a remote lab.
  • Resetting a remote lab.
  • Deploying a new image to a remote lab.
  • Powering on a remote lab.
  • Installing ReadyTech lab tools.

To view the log of a remote lab, follow the steps below:

  1. Click the Lab tab on the navigation bar, then click System log...
  2. Select the check-box next to each student whom you want to view their remote labs log.
