Viewing the system log

You can use the System log command to access the log for a remote system. The system log is useful to troubleshoot issues such as deploying or resetting images.

To view the log of a system:

  1. Click Lab from the top menu bar then click System log...
  2. Select the check-box next to each user whom you want to view their systems log.
