Actions menu

The Actions menu under the Lab tab allows you to perform various tasks to the lab such as:

  • Block user access to certain areas of the portal.
  • Reboot the remote systems.
  • Transfer files to and from the remote systems.
  • Reset the remote systems to their original states.
  • Deploy a new image to the remote systems.
  • Clear the activation information of an access code.
  • Change a users access code.
The Actions menu can be found under both the Lab tab and the Desktop tab. Lab actions can be run against multiple systems, and are available only to instructors. The Desktop Actions menu is available to all users and actions run affect only the system a user is connected to.
