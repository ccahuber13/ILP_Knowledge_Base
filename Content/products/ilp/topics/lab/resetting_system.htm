Resetting the remote systems

You can use the Reset system command to restore a remote system to its initial, preconfigured state. Be aware that using Reset system reinstalls the original software image and wipes out any software and data on the system; any work you have done up to this point will be lost.

To reset a remote system:

  1. Click Lab from the top menu bar then click Reset system...
  2. Select the check-box next to each user whom you want to reset their system.
  3. Click Reset from the Reset system prompt.
    Click the Lab tab then select Overview to see the status of the reset system.
