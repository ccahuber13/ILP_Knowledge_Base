Updating remote lab deployment files

Use Update deployment file… to update the rt_deployment.txt file with the latest deployment information for your event. Deployment files are located on each remote lab in file directory C:\rt_deployment.txt.

Deployment files include the following information:

  • Access code
  • Event ID
  • Activation information
  • Deployment and System information
  • User role

To update deployment files, follow the steps below:

  1. Click the Lab tab on the navigation bar.
  2. Click Actions, then click Update deployment file....
  3. Select the check-box next to each student whom you want to update their deployment file.
  4. Click OK.
