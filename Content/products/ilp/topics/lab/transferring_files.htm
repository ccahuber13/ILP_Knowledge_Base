Transferring files to and from the remote labs

Use Transfer file to remote systems… or Collect files from remote systems… to transfer files directly to and from the remote labs.

Watch the video below or continue reading to learn how to transfer files to and collect files from the remote labs.

Transferring files

To transfer multiple files, first zip them into a single file and transfer the zipped file. Find files transferred to the remote labs in file directory c:\ReadyTech\Inbox.

To transfer a file to the remote labs, follow the steps below:

  1. Click the Lab tab on the navigation bar, then click Transfer file to remote systems...
  2. Select the check-box next to each student whom you want to transfer files to.

Collecting files

Ask students to place files to be collected in file directory c:\ReadyTech\Inbox on their remote labs. When collecting files from multiple students, they are zipped into a single file with multiple directories named after each student's access code.

To collect files from the remote labs, follow the steps below:

  1. Click the Lab tab from the navigation bar.
  2. Click Actions, then click Collect files from remote systems....
  3. Select the check-box next to each student whom you want to collect files from.
  4. Click OK.
  5. Save the zipped download when prompted to a location on your local system. Note that some browsers automatically save the file to a default downloads folder.
