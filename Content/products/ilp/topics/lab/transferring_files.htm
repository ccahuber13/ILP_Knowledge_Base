Transferring files

You can transfer files directly to and from the remote systems. Files transferred from the actions menu do not appear under the Materials tab in the portal. Instead, transferred files can be found in directory c:\ReadyTech\Inbox and Outbox.

Watch the video below to learn how to transfer files to and collect files from the remote lab.

Transfer file to remote systems...

To transfer files to the remote system:

  1. Click Lab from the top menu bar then click Transfer file to remote systems...
  2. Select the check-box next to each user whom you want to transfer a file to.

Collect files from remote systems...

To collects files from the remote system:

  1. Click Lab from the top menu bar.
  2. Click Actions then click Collect files from remote systems....
  3. Select the check-box next to each user whom you want to collect files from.
  4. Click OK.
  5. Save the zipped download to a location on your local system.
