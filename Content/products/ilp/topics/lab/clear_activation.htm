Clearing activiation

Clearing activation for an access code removes the name currently associated with the code. Once activation is cleared, the access code can be re-activated with a new name. This can be useful in situations where two classes are being taught in the same day using the same remote systems. An instructor can clear access codes and reset the remote systems for new students to activate.

  1. Click Lab from the top menu bar then click Clear activation...
  2. Select the check-box next to each user whom you want to clear activation.
