Blocking user access

Use Block user access… to block student access to specific areas of the Instructor-Led Portal (ILP) such as the portal itself, the Materials page, or the Lab page.

To block user access, follow the steps below:

  1. Click the Lab tab on the navigation bar.
  2. Click Actions, then click Block user access.
  3. Click the check-box next to each student whom you want to block access to a specific area of the portal.
    • Select Portal to block a student from logging in to the ILP.
    • Select Materials to hide the Materials tab on the navigation bar.
    • Select Labs to hide the Lab tab on the navigation bar.
  4. Click Save.
