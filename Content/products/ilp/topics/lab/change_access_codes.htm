Changing access codes

Use Change access codes… to change access codes for an event. Changing access codes are useful to revoke a student's access to the portal, or for using a single event to teach separate morning and afternoon classes.

To change access codes, follow the steps below:

  1. Click the Lab tab on the navigation bar, then click Change access codes...
  2. Select the check-box next to each user whom you want to change access codes.
  3. Click OK.
