Changing access codes

You can use this feature to change one or more access codes. You might use it to revoke a student's access to the portal. Alternatively, it can also be used to change access codes form a morning class, to an afternoon class.

  1. Click Lab from the top menu bar then click Change access codes...
  2. Select the check-box next to each user whom you want to change access codes.
