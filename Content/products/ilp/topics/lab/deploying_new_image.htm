Deploying a new image to the remote systems

You can use the Deploy new image command to install a new software image on the remote system. This process will wipe out all software and data on the remote system; any work you have done up to this point will be lost.

To deploy a new image:

  1. Click Lab from the top menu bar then click Deploy new image...
  2. Select the check-box next to each user whom you want to deploy a new image.
  3. Select an image from the image selection drop-down list.
  4. Click OK.
  5. Click Deploy on the Deploy new image prompt.
    Click the Lab tab then select Overview to see the status of the reset system.
