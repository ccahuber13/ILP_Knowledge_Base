Connecting to a student remote system

From the Lab page, you can connect directly to a students remote system.

To connect to a student remote system:

  1. Select a student from the drop-down list below the top menu bar.
  2. Click the remote desktop thumbnail to connect to the lab.

Switch between multiple remote systems

To switch between your remote desktop and the students remote desktop:

  1. Click Desktop from the top menu bar.
  2. Select a user from the list of systems you are currently connected to.

Disconnect from a remote system

To disconnect from a student remote system:

  1. Click Desktop from the top menu bar.
  2. Select a user from the list of remote systems you are currently connected to.
  3. Click Disconnect. Alternatively, click Disconnect all to close all remote connections.
The actions menu and commands from the Desktop tab affect only the system you are currently viewing. If you'd like to perform Actions on multiple systems at once, use the Actions menu from the Lab tab.
