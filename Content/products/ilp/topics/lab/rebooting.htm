Rebooting the remote systems

From the Lab actions menu, you can choose to soft reboot or hard reboot a single or multiple student systems.

  • Soft reboot - Attempts to software reboot a system. This functions similar to going to Start > Shutdown > Restart in Windows.
  • Hard reboot - Attempts to restart the remote system by powering it off and on.
Always attempt a soft reboot first. Only use hard reboot if the remote system hangs to prevent possible data loss.

To reboot a remote system:

  1. Click Lab from the top menu bar.
  2. Click Actions then click Soft reboot....
  3. Select the check-box next to each user whom you want to reboot their system.
  4. Click OK.
The Status icon will change to down, and then back to up when the system as finished restarting.
