Lobby

The Lobby page is a community meeting point for an Instructor-Led class. From the Lobby page a user can:

  • Find class event information
  • View other user profiles
  • Edit profile information
  • View a map of other participant locations
  • View training videos for the Instructor-Led Portal
  • Find notices and alerts from ReadyTech
