Desktop actions menu

Run actions found under the Desktop Actions menu to manage the remote system you are currently connected to.

Note that the Actions menu can be found under both the Lab tab and the Desktop tab. Lab actions can be run against multiple systems, and are available only to instructors. The Desktop Actions menu is available to all users and actions run affect only the system a user is connected to.

  • See Actions menu for more information.
Students are limited to the following actions:

  • Soft reboot
  • Hard reboot
  • Power on
  • System log
  • View action history
