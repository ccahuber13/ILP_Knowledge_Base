Enabling password protection

For additional security, users can set a password associated with their access code login.

To enable password protection, follow the steps below:

  1. Click your name from the user menu in the top right-hand corner of the portal.
  2. Click Enable password protection.
  3. Enter a Password and then confirm your password.
    There are no specific character limitations or requirements to creating an access code password.
  4. Click Save.

To clear a password associated with an access code, see clearing activation.
