Enabling the on-screen keyboard

You can enable an on-screen virtual keyboard in situations where the remote system does not respond to certain physical keyboard commands, or if accessing the Instructor-Led Portal from a mobile client.

To enable the on-screen keyboard, follow the steps below:

  1. Click your name from the user menu in the top right-hand corner of the portal.
  2. Click Settings.
  3. Click On-screen keyboard.
  4. (Optional) Select to show the keyboard on mobile clients only, Always display, or never display on the navigation bar.
  5. (Optional) Select a Keyboard button action.
    • Use Display remote native keyboard to use a virtual keyboard within the remote system.
    • Use Display local Javascript keyboard to use a virtual keyboard for your local system.
  6. (Optional) Select a Keyboard layout for different languages.
  7. Click Save.

On the navigation bar, click the keyboard icon to open the on-screen keyboard.
